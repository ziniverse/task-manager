Task Manager
A simple, interactive web-based task manager to help you organize your tasks efficiently. This project, available here, is designed to streamline task management with an intuitive interface, allowing users to add, update, and delete tasks with ease.

Features
Add Tasks: Quickly add new tasks with a title and description.
Edit Tasks: Modify existing tasks to update their content or status.
Delete Tasks: Remove tasks that are no longer needed.
Responsive Design: Accessible on both desktop and mobile devices.
Getting Started
Clone the repository:

bash
Copy code
git clone https://github.com/ziniverse/task-manager.git
cd task-manager
Open index.html in your browser to view the task manager.

Alternatively, access the live version here.

Usage
Add a Task: Enter the task name and description, then click "Add Task."
Edit a Task: Click on an existing task to modify its details.
Delete a Task: Select the delete icon next to the task to remove it.
Project Structure
index.html - The main HTML page.
style.css - Contains the styling for the app.
script.js - JavaScript functionality for managing tasks.
Built With
HTML - For structuring the web page.
CSS - For styling and layout.
JavaScript - For functionality and task management.
Future Enhancements
Task Filtering: Ability to filter tasks by status or category.
Reminders: Option to set reminders for tasks.
Persistence: Save tasks locally, allowing data retention even after page refresh.
